WTWR (What to Wear?): Back End
The back-end project is focused on creating a server for the WTWR application. Through this project, I gained a deeper understanding of working with databases, setting up security and testing, and preparing for deployment. The eventual goal is to create a server with an API and user authorization.

Technologies and Techniques Used
Node.js — A JavaScript runtime built on Chrome's V8 JavaScript engine.

Express.js — A minimal and flexible Node.js web application framework.

MongoDB — A NoSQL, document-based database.

Mongoose — An Object Data Modeling (ODM) library for MongoDB and Node.js.

Validator — A library of string validators and sanitizers (used for URL validation).

Nodemon — A tool that helps develop Node.js based applications by automatically restarting the node application when file changes in the directory are detected.

ESLint — A tool for identifying and reporting on patterns found in ECMAScript/JavaScript code, configured with Airbnb Base and Prettier for consistent code style.

REST API — Implementation of routes and controllers for users and clothing items.

MVC Pattern — Organized the code into Models, Views (represented by JSON responses), and Controllers.
## Running the Project
`npm run start` — to launch the server 

`npm run dev` — to launch the server with the hot reload feature
